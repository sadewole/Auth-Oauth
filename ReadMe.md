## Installation

Clone repo

```sh
git clone https://github.com/sadewole/Crispy-Munch-v2-.git
cd Crispy-Munch-v2-
```

Install the dependencies:

> Installs server dependencies ::
> `yarn install`

> Installs client dependencies ::
> `cd client && yarn install`
